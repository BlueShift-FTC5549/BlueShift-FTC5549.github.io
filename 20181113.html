BLUE SHIFT

2018/11/14 - SWEEPER PROTOTYPE, BLUE SHIFT SOFTWARE 0.2.0

Today, we worked on creating an effective intake method that can that can easily sweep blocks and balls into an inner tray. We worked to leave as much room as possible for a future sorting method. However, the sweeper placement was too high, so we are planning to switch to horizontal extensions next meeting.

Furthermore, the programming team created three programs to individually turn the Robot using the Rev Hub's IMU. This was released on our team Github repository.

Last but not least, version 0.2.0 of the Blue Shift software was released on GitHub. We added an auto encoder drive, updated our code for the IMU, and added some utilities for our robot.
