BLUE SHIFT

2018/11/12 - WHEEL BASE, CHARGING STATION, AND TEAM GITHUB

Today we finalized our wheel base and started brainstorming on an intake method for the robot to sweep in minerals with. The wheels base isn't of much importance yet, since we can always go back and revise it if anything goes wrong. The intake method was our main focus today.

Additionally, we updated our charging station by replacing the batteries with our dorm mains supply to improve compatibility with other devices. We've had the charging station since last year and it is where everything goes for extra juice.

Our team also created an official Github account and migrated our exsisting website to the new account. The github will not only host our website but also the code we use for our robot and also 3D-printer files used for our marker, which we haven't chosen a design for yet. We set up the Github repository in all the programmers' computers.
